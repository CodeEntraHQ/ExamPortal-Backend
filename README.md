## How to Run the Backend

### 1. Prerequisites

- Node.js (v18 or above)
- npm (comes with Node.js)
- PostgreSQL (local or cloud)
- Docker (optional)

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Download from drive

### 4. Start the Server

```bash
npm run dev
```
