## How to Run the Backend

### 1. Prerequisites

- Node.js (v18 or above)
- npm (comes with Node.js)
- PostgreSQL (local or cloud)
- Docker (optional)

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Server

```bash
npm run dev
```

### 4. Running tests

```bash
npm test
```

### 5. Running coverage

```bash
npm run coverage
```
